# F1APM Portfolio

Personal portfolio for Shaurya Sethi — PM, writer, and host of the F1APM podcast.

**Live site:** https://shauryasethi.com

## Tech stack

- Vite + React + TypeScript
- shadcn/ui + Tailwind CSS
- Deployed on Vercel (auto-deploys on push to `main`)

## Local development

```sh
git clone https://github.com/shaurya133/f1apm-portfolio
cd f1apm-portfolio
npm install
npm run dev
```

Open [http://localhost:8080](http://localhost:8080).

## Deploying

Push to `main` — Vercel picks it up automatically.
